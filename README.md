# AI-Daily-Bot
Automatically fetches the latest AI news from YouTube / TechCrunch / Twitter every day at 20:00 Beijing time, translates and summarizes them via Groq LLM, and posts the final digest to your any social media have API e.g.Twitter / Instagram.   > Runs entirely on GitHub Actions free tier. 每日北京时间 20:00自动抓取最新 AI 资讯，加工成中文日报，并同步发布，可发图文。
